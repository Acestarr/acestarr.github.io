A. CSP提高组模拟赛T1【中山-2018.8.14A组-T1】区间

Description

有一个 nnn 个数的序列，一开始所有的数都是 000，每次可以将一个区间 [l,r](l≤r)[l,r](l\le r)[l,r](l≤r) 内的数 +1+1+1，求到达最终状态的最少操作次数。

Solution

考场上想了一个神奇的单调栈做法，对于 aia_iai​，如果 ai>stktopa_i > stk_{top}ai​>stktop​，就把 [stktop+1,ai][stk_{top}+1,a_i][stktop​+1,ai​] 间的数全部入栈，每个数都加一个答案区间，左端点为 iii，否则一直弹栈，并将栈中对应区间的右段点设为 i−1i-1i−1。

讲评的时候发现他们都是写的 O(nlog⁡n)O(n\log n)O(nlogn) 的，维护区间最小值，分治。

不过因此我拿到了最优解（42ms）

B. CSP提高组模拟赛T2【中山-2018.8.14A组-T3】树

Description

给定一棵 nnn 个点的树和 mmm 个询问，对于每个询问 ui,viu_i,v_iui​,vi​，求从 uiu_iui​ 走到 viv_ivi​ 的期望步数。

在每个点都会等概率的走向其连着的点。

Solution

考场上打完暴力就一直在做这道题，写了个 O(n2)O(n^2)O(n2) 的，但是最后只有 10pts10pts10pts qwq

下来对比了一下错误输出，不一样的值我输出都是 000。

正解做法太神仙了

设 fif_ifi​ 表示从 iii 走到 faifa_ifai​ 的期望步数，gig_igi​ 表示从 faifa_ifai​ 走到 iii 的期望步数。

做个前缀和，sfisf_isfi​ 表示从 iii 到根的 fif_ifi​ 的和，sgisg_isgi​ 表示从 iii 到根的 gig_igi​ 的和。

最后答案就是 sfu−sfw+sgv−sgwsf_u-sf_w+sg_v-sg_wsfu​−sfw​+sgv​−sgw​，其中 www 为 lca(u,v)lca(u, v)lca(u,v)

考虑 fuf_ufu​ 怎么算

  1. 直接从 uuu 走到 faufa_ufau​：+1+1+1
  2. vvv -> uuu -> faufa_ufau​：+1degu∑(fv+fu+1)+\dfrac{1}{deg_u}\sum (f_v+f_u+1)+degu​1​∑(fv​+fu​+1)

因此 fu=1+1degu∑(fv+fu+1)f_u = 1+\dfrac{1}{deg_u} \sum (f_v+f_u+1)fu​=1+degu​1​∑(fv​+fu​+1)

整理可得 fu=1+∑(fv+1)f_u=1+\sum (f_v+1)fu​=1+∑(fv​+1)

用 sfusf_usfu​ 记下 ∑(fv+1)\sum(f_v+1)∑(fv​+1)，后面算 ggg 要用（注意这里 sfsfsf 和上面的不一样）

再来看看 gug_ugu​ 怎么求

  1. fafaufa_{fa_u}fafau​​ -> faufa_ufau​ -> uuu：+1+gfau+1+g_{fa_u}+1+gfau​​
  2. 从 uuu 的兄弟走到 faufa_ufau​，再走到 uuu：+sffau−(fu+1)+1+sf_{fa_u}-(f_u+1) + 1+sffau​​−(fu​+1)+1（总的减去从 uuu 走上去的）

因此 gu=sffau−(fu+1)+1+gfau+1g_u=sf_{fa_u}-(f_u+1)+1+g_{fa_u}+1gu​=sffau​​−(fu​+1)+1+gfau​​+1

这道题就做完了。

C. CSP提高组模拟赛T3【中山-2018.8.15A组-T2】做运动

Description

给定一张 nnn 个点，mmm 条边的无向图，每条边有两个权值 t,ct,ct,c，求从 sss 到 ttt，在保证经过的边中最大的 ttt 最小的前提下，使得 ∑ti×ci\sum t_i\times c_i∑ti​×ci​ 最小。

Solution

考场上不知道怎么犯傻就写了个二分，复杂度 O(nlog⁡2n)O(n\log^2n)O(nlog2n)，T飞了，只有 30pts30pts30pts

其实先按 ttt 排个序，然后用并查集维护一下，可以直接求出 ttt 的最小值。

然后把符合条件的边加进去，跑个最短路就行了。
